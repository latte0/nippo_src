課題終わり。大したことはしていないからER図とスクリプトだけ置いておく。

```
[Album]
*id
name
year
+album_artist_id
+genre_id

[Artist]
*id
name

[Genre]
*id
name

[Track]
*id
name
+artist_id
+album_id

Album *--1 Artist
Album *--1 Genre
Track *--1 Album
Track *--1 Artist
```

これを`erd`に渡すとこうなる。

いいね！

いいか。昼が遅くなったからカロリーメイトとグリーンスムージーで軽く済まそうと
している。持つといいな。
